# Soccer Stat

⚽ **Футбольная статистика** — SPA приложение на React (TypeScript + Vite) для просмотра футбольных лиг, команд и календаря матчей. Проект разработан как тестовое задание в SimbirSoft.

## 🌟 Особенности проекта

* Полностью соответствует **Pixel-Perfect** дизайну в Figma.
* Использован **API `football-data.org`** для получения реальных данных (Лиги, Матчи, Команды).
* **Мобильная адаптивность** (Media Queries) — сайт корректно отображается как на десктопе, так и на телефонах.
* **Кастомные элементы UI:**
  * Улучшенная пагинация (показывается только если страниц > 1).
  * Date Picker с кастомной иконкой календаря.
  * Search Bar с кастомной лупой.
* Полный отказ от тяжелых UI-библиотек (в пользу скорости работы, весь UI переписан на чистом CSS).
* **Оптимизация работы с API:**
  * Кеширование запросов (In-Memory Cache на 5 минут).
  * Дедупликация (In-Flight Request Deduplication) для предотвращения ошибки `429 Too Many Requests`.

## 🚀 Установка и запуск (Локально)

1. Клонируйте репозиторий:
   ```bash
   git clone <URL>
   cd soccer-stat
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Создайте файл `.env` в корне проекта (рядом с `package.json`) и добавьте свой API-ключ от `football-data.org`:
   ```env
   VITE_FOOTBALL_API_KEY=ваш_api_ключ_здесь
   ```
   > Вы можете получить бесплатный ключ на сайте [football-data.org](https://www.football-data.org/).

4. Запустите dev-сервер:
   ```bash
   npm run dev
   ```

5. Откройте в браузере: [http://localhost:5173/](http://localhost:5173/)

## 📝 Структура проекта

```text
src/
├── components/       # Переиспользуемые UI-компоненты (Layout, Pagination, DateInput)
├── pages/            # Страницы приложения (Лиги, Календарь лиги, Команды, Календарь команды)
├── services/         # Настройка Axios API-клиента (кеш + вызовы)
├── types/            # TypeScript интерфейсы для API
├── utils/            # Вспомогательные функции (форматирование дат)
├── App.tsx           # Корневой роутер приложения
└── main.tsx          # Точка входа
```

## 🧩 Комментарии к реализации

* В бесплатном тарифе API (Free Tier) действует лимит: **10 запросов в минуту**. Чтобы минимизировать вероятность блокировки (ошибка 429), был убран `React.StrictMode` из `main.tsx` и написан умный кэшер запросов внутри `api.ts`.
* Вместо `Ant Design` используется полностью собственный UI на чистом CSS (Grid, Flexbox), что значительно уменьшило размер финального бандла.
